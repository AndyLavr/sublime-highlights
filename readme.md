# sublime-highlights
Syntax highlighting for Sublime Text 3.

## Supported formats

* knf - KNF format
* man - man page format
* mockka - Mockka rule format
* rbdef - RBBuild defenition file format
* spec - RPM Spec format
* upstart - Upstart format
